### Installation
If you just want to install the dotfiles:
```bash
$ curl -Lks https://raw.githubusercontent.com/yochem/dotfiles/master/.install/dotinstall | /bin/bash
```
and installing things brew, pip, changing mac defaults etc:
```bash
$ cd .install/
$ bash install
```
